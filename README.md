# 🌙 StayUpAllNight

StayUpAllNight è una piattaforma web progettata per offrire un’esperienza semplice e immediata nella scoperta e nell’organizzazione di attività notturne, eventi e momenti da vivere fuori dagli schemi tradizionali.

---

## 🚀 Panoramica

Il progetto nasce con l’obiettivo di rendere più accessibile e centralizzata la ricerca di esperienze notturne, offrendo un’interfaccia essenziale, veloce e intuitiva.

StayUpAllNight si propone come uno strumento moderno per chi vuole vivere la notte in modo diverso, senza perdere tempo tra fonti frammentate.

---

## ✨ Funzionalità principali

* 🔎 Esplorazione di attività ed esperienze notturne
* ⚡ Interfaccia rapida e minimal
* 📱 Design responsive e ottimizzato per mobile
* 🌐 Accesso diretto via browser

---

## 🛠️ Stack Tecnologico

> Nota: aggiornare questa sezione in base alle tecnologie effettivamente utilizzate.

* Frontend: HTML, CSS, TypeScript
* Framework/Librerie: Vite & React (TypeScript)
* Backend: Supabase, Resend, Google Cloud
* Hosting: Vercel

## 📦 Installazione

Clona la repository:

```bash
git clone https://github.com/tuo-username/stayupallnight.git
cd stayupallnight
```

Installa le dipendenze:

```bash
npm install
```

Avvia il progetto in locale:

```bash
npm run dev
```
